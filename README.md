> **Function to determine if something is a string type**

## Install

```sh
npm install --save @chriscodesthings/stub-repo
```

## Use

```js
import stubRepo from '@chriscodesthings/stub-repo';

console.log(stubRepo("hello world!"));
// => true
```

## Syntax

```js
stubRepo(var);
```

### Parameters

- *var*: any

### Return Value

Returns something probably.
